# 架構與資料流程

```text
DailyScheduleCoordinator
  -> DailyJobService
     -> CrawlerRegistry -> CnyesTechnicalAlignmentCrawler -> Parser
     -> SqliteYiHeLeeRepository（四個必要批次驗證後單一交易）
     -> ExcelWorkbookService（讀取持股）
     -> StrategyEvaluationService
     -> SqliteYiHeLeeRepository（持股快照及通知）
     -> ExcelWorkbookService（覆寫輸出頁籤並儲存）
     -> WinFormsUserInteraction -> 中央通知視窗
```

## 重試層級

- 爬蟲短重試：網路、瀏覽器、DOM 暫時錯誤。
- Excel 短重試：RPC 呼叫被拒、Excel 忙碌、正在編輯儲存格。
- 排程長重試：網站尚未更新當日資料、短重試耗盡等可重試狀態。
- 設定無效、必要 Provider 不存在、Excel 唯讀／受保護檢視、輸出頁籤受保護等不可重試錯誤，需要人工修正。
- 非必要來源必須集中／店頭兩個市場都成功才會納入正式批次；其中一個市場失敗時整個非必要來源略過。

## 權限

預設與使用者同權限執行，以連接一般權限的 Excel。可從系統匣選單要求 UAC 提升，但提升後 Excel 也必須以相同權限啟動。
