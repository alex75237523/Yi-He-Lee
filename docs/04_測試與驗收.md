# 測試與驗收

## 自動測試

- MA5、MA20、MA120 任一成立即通知。
- 三項都不成立時不通知。
- 技術資料缺少時列為無法判斷。
- 綠色 `#92D050` 與文字標記排除。
- 一般黃色持股不可誤排除。
- 固定來源及 13:35 不可被關閉。
- 鉅亨表格完整解析、缺表頭失敗、頁面日期擷取。

## Windows 人工驗收

1. Excel 未開啟：應中央顯示失敗，Log 有明確原因，排程可重試。
2. Excel 正在編輯儲存格：短重試後成功，或達上限顯示失敗。
3. Excel 有 Modal 對話框：不得卡死；關閉後重新執行可成功。
4. 活頁簿唯讀／受保護檢視：不得寫入。
5. 輸出頁籤受保護：不得寫入，原檔不得被破壞。
6. 頁面顯示昨日日期：SQLite 正式技術表不得出現今日資料。
7. 同一天重跑：技術資料、持股快照、通知不得重複。
8. 綠色股名列：不得進入判斷清單。
9. 沒有「自持股」標題但表頭完整的區塊：必須納入。
10. 含「出場價」的已出場表格：不得納入。
11. 中央通知：需在螢幕中央，完整顯示客戶、股票及觸發均價。
12. 系統匣：關閉中央視窗後程式仍在右下角運行。
