# Yi He Lee－完整需求與目前狀況

## 1. 專案目的

本程式用於每日台股收盤後，自動取得鉅亨網技術指標，與客戶 Excel 活頁簿中持股的「現價」欄位（外部 DDE）比較，將符合條件的客戶與股票彙整到 Excel，並在螢幕中央提醒使用者。（2026-07-11 更正：比較基準由「進場價／平均價」改為「現價」。）

## 2. 使用者實際工作方式

- 使用者平常會開著 Excel 工作。
- 使用者可能在其他客戶頁籤操作，但不會操作程式輸出的 `每日五日均價策略` 頁籤。
- 程式必須在 Excel 開啟狀態下寫入指定頁籤。
- 程式常駐右下角系統匣，不需要一直顯示主視窗。
- 每日固定在台北時間 13:35 自動執行，也要能從系統匣手動執行。

## 3. 目前程式狀況

- 已有一套 .NET 8 原始碼與 Solution。
- 原始碼包含 App、Application、Infrastructure、Domain、Tests 等分層。
- 使用者目前尚未成功編譯。
- Claude Code 的工作是先讀懂現有程式、實際 Build、修正錯誤，再完成實機驗證。
- 不允許直接丟掉現有程式後重寫。

## 4. 最終技術決定

| 項目 | 最終決定 |
|---|---|
| UI | .NET 8 Windows Forms |
| 執行型態 | Windows 系統匣常駐程式 |
| 資料庫 | SQLite，完全離線、可攜式 |
| SQLite 套件 | Microsoft.Data.Sqlite |
| Excel | Microsoft Office Interop Excel |
| 爬蟲 | 以 Provider／Parser 擴充，現有來源可用 Playwright／HttpClient |
| 排程時區 | Asia/Taipei |
| 每日時間 | 13:35 |
| 通知位置 | 螢幕正中央 |
| 管理員 | 預設一般權限，可選擇以管理員重新啟動 |

## 5. 已取消決定

- SQL Server Express LocalDB。
- MSSQL MDF／LDF。
- Microsoft.Data.SqlClient。
- `%LOCALAPPDATA%` 作為唯一資料位置。

最終要求是整個程式資料夾可搬到另一台已具備執行環境的 Windows 電腦使用，因此設定、SQLite、Log、備份應放在程式資料夾下。

## 6. Excel 頁籤規則

- `總表`：排除，不掃描持股。
- `每日五日均價策略`：輸出頁籤，排除，不掃描持股。
- 其他頁籤視為可能的客戶頁籤。
- 有些頁籤沒有「自持股」標題，仍要辨識。
- 以表頭結構找出有效持股：代號／代碼、股名／名稱、現價。（2026-07-11 更正：原「進場價／平均價」為口誤。）
- 已出場區塊通常包含 `出場價` 或 `出場日`，不納入。
- 特定顏色與文字標記的 LIST／列不納入；文字為主、顏色為輔。
- 股票代碼必須保留前導零及字尾。

## 7. 技術指標來源

固定兩頁：

1. `a_technical4.aspx`：股價多頭排列。
2. `a_technical5.aspx`：股價空頭排列。

每頁都要擷取集中市場及店頭市場完整資料。頁面顯示日期必須與台北當日一致。

## 8. 判斷規則

任一成立即通知（2026-07-11 更正：比較基準由「進場價／平均價」改為 Excel「現價」欄位）：

- MA5 <= 現價。
- MA20 <= 現價。
- MA120 <= 現價。

MA60 只保存與顯示，不參與觸發。

「現價」欄位串接外部 DDE（看盤軟體），必須小心判讀：

- 儲存格為 Excel 錯誤值（#N/A、#NAME? 等，常見於看盤軟體未開啟或 DDE 未連線）、空白、0、負數或無法解析的文字時，該持股一律不得進行均線判斷。
- 不得以收盤價、昨日現價或任何其他數值代替。
- 這些持股必須列入「現價異常」清單，於螢幕中央結果視窗與 Log 明確告知使用者原因。

## 9. 結果處理

- 先將當日爬蟲資料完整驗證後寫入 SQLite。
- 再讀取 Excel 有效持股。
- 再進行策略比對。
- 再把結果寫到 `每日五日均價策略`。
- 最後在螢幕中央顯示彙整清單。
- 未找到技術資料的持股列入「無法判斷」。
- 不得使用昨天資料補今天。

## 10. 使用者已確認的防呆

- 執行前提示先按 Enter／Esc。
- 關閉 Excel 對話框。
- 執行中不要關閉 Excel。
- 不要改名或刪除輸出頁籤。
- Excel 忙碌要短暫重試。
- 最終失敗程式要知道，寫 Log、記錄 Job，並中央顯示失敗。
- 可重試錯誤依排程再次執行。
- 不可重試錯誤要求人工修正。
- 不可無限重試。

- 中央通知與系統匣正常。
- 失敗、重試、Log、Job 紀錄正常。
